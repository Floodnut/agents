# Local Knowledge Base

Reusable notes for project and infrastructure work.

## Index format
- Keep `index.md` as the entry point.
- Add one bullet per note with title, path, tags, and a short summary.

## Note format
- Title
- Summary
- Context
- Details
- Commands
- References
- Tags

## Triggers
Update this knowledge base after non-trivial troubleshooting, recurring findings, or reusable design/operation decisions.
